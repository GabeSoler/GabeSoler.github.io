About

I am Gabriel Soler; I work full-time as a psychotherapist and have been learning to code for the last few years. I have found it fascinating to learn programming logic and deepen my understanding of cybernetics, a way of thinking I have seen in many theories before and have always loved.

If you put those lenses, everything can be understood as an information system. That helps me move between paradigms quickly as I find patterns between them.

I have been using my process of learning programming to develop my model of psychotherapy, inspired in part by my engagement with the logic of computer science, which I have applied to the field of psychotherapy. At the same time, I try to keep a firm ground in my practice and experience, as I fear an excess of abstraction can also become a problem.

Web development allows me to publish and try these ideas into more dynamic forms quickly. I have been using the Django web framework combined with Bootstrap. Slowly, I feel more confident using these tools and fluent when I get what I want.
